# Retail Sales Management System - Frontend

Modern React application for retail sales management with advanced filtering, sorting, and search capabilities.

## Tech Stack

- React 18
- Vite
- Ant Design 5
- React Router
- Axios

## Setup Instructions

1. Navigate to frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Application will run on http://localhost:3000

## Build for Production

```bash
npm run build
```
